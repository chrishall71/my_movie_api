Documentation

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

The table below describes each endpoint of this API and how to access it.

Description URL HTTP method Request body data format Response body data format
Get a list of all movies /movies GET None JSON object holding data about all movies. Structure:

{Id:"5e1fa2c51955aa2449c5fdef"},"Title":"Toy Story 4","Description":"Toy Story 4 is a 2019 American computer-animated comedy film produced by Pixar Animation Studios for Walt Disney Pictures. It is the fourth installment in Pixars Toy Story series.","Genre":{"Name":"animation","Discription":"Animation is a method in which pictures are manipulated to appear as moving images."},"Director":{"Name":"Josh Cooley","Bio":"Joshua Cooley (born May 23, 1980) is an American animator, screenwriter, director and voice actor.","Birth":"1980","Death":""},"ImagePath":"toystory_4.jpg","Featured":false}

Get data about a single movie by title /movies/:Title GET None JSON object holding data about a single movie. Structure same as above.
Get data about a movie genre /movies/genres/:Name GET None JSON object holding data about a movie genre, containing name and description. Structure:

{Name: "Drama", Description: "Movie with serious undertones, etc."}

Get data about a director /movies/directors/:Name GET None JSON object holding data about a director. Structure:

{name: "Michael Curtiz", bio: "Hungarian-born American Director, etc.", birth: "1886", death: "1962"}

Add a new user to the registry. /users POST JSON object holding data about the user to add. Structure:

{ id: 1, Name: John Doe, Username: "Johndoe81", Password: "pw123", Email: "john@gmail.com", Birthday: "12/20/66", FavoriteMovies:["5e1fc18a1955aa2449c5fdf8"]}

JSON object holding data about the user that was add. Structure:

{ id: 1, Name: John Doe, Username: "Johndoe81", Password: "pw123", Email: "john@gmail.com", Birthday: "12/20/66", FavoriteMovies:["5e1fc18a1955aa2449c5fdf8"]}

Update a user's information /users/:Username PUT JSON object holding data about the user to add. Structure:

{ id: 1, Name: John Doe, Username: "Johndoe81", Password: "pw123", Email: "john@gmail.com", Birthday: "12/20/66", FavoriteMovies:["5e1fc18a1955aa2449c5fdf8"] }

A message indicating the user's information was updated.
Add a movie to the user's favorites list /users/:Username/movies/:movieID POST None A message indicating the movie title was added to the user's favorites list.
Delete a movie from the user's favorites list /favorites/:Username/Movies/:movieID DELETE None A message indicating the movie title was deleted from the user's favorites list.
Remove a user from the registry. /users/:Username DELETE None A message indicating the user was removed from the registry.
